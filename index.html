AWS 실습 사이트

AWS 서비스 실습 페이지

AWS의 데이터베이스, 서버리스 컴퓨팅, 보안 암호 관리 서비스를 간단하게 확인할 수 있습니다.

AWS RDS

MySQL, PostgreSQL 등 관계형 데이터베이스를 손쉽게 생성하고 운영할 수 있는 관리형 데이터베이스 서비스입니다.

AWS Lambda

서버를 직접 관리하지 않고 이벤트가 발생할 때 코드를 실행하는 서버리스 컴퓨팅 서비스입니다.

Secrets Manager

데이터베이스 비밀번호와 API 키 같은 민감한 정보를 안전하게 저장하고 관리하는 서비스입니다.
